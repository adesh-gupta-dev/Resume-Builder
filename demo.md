# Resume Builder Demo Guide

## Quick Start Demo

This guide will walk you through creating a professional resume using the Resume Builder application.

### Step 1: Launch the Application

1. Open your terminal and navigate to the Resume-Builder directory
2. Run `npm start` to launch the development server
3. Open your browser and go to `http://localhost:3000`

### Step 2: Start Building Your Resume

1. **Click "Create Resume"** on the homepage
2. **Click "üìù Load Sample Resume"** to see how it works with example data
   - This will populate all fields with sample information
   - You can then edit the data to match your own information

### Step 3: Complete Each Form Step

#### Step 1: Personal Information
- **First Name**: Enter your first name
- **Last Name**: Enter your last name  
- **Email**: Your professional email address
- **Phone**: Your contact number
- **Location**: City, State (e.g., "San Francisco, CA")
- **LinkedIn**: Your LinkedIn profile URL (optional)

#### Step 2: Professional Summary
- Write a compelling 2-4 sentence summary of your professional experience
- Focus on your key skills, achievements, and career objectives
- Keep it concise and impactful

#### Step 3: Work Experience
- **Job Title**: Your position title
- **Company**: Company name
- **Location**: Work location
- **Start Date**: When you started (month/year format)
- **End Date**: When you ended (leave empty if current)
- **Current**: Check if this is your current job
- **Description**: Key responsibilities and achievements
- Use the "+ Add Experience" button to add more jobs

#### Step 4: Education & Skills
- **Degree**: Your degree (e.g., "Bachelor of Science in Computer Science")
- **School**: University or institution name
- **Graduation Date**: When you graduated
- **GPA**: Your GPA (optional)
- **Skills**: List your technical and soft skills, separated by commas
- Use the "+ Add Education" button to add more education entries

#### Step 5: Choose Template
- **Classic Professional**: Traditional design for corporate jobs
- **Modern Minimalist**: Contemporary two-column layout
- **Creative Portfolio**: Eye-catching design with gradients

### Step 4: Preview and Export

1. **Preview**: Review your resume in the selected template
2. **Download PDF**: Click "Download PDF" to save your resume
3. **Print**: Use the print button for physical copies

## Tips for Success

### Content Tips
1. **Keep it concise**: Aim for 1-2 pages maximum
2. **Use action verbs**: Start bullet points with strong action words
3. **Quantify achievements**: Include numbers and percentages when possible
4. **Tailor to job**: Customize content for specific positions
5. **Proofread carefully**: Check for spelling and grammar errors

### Technical Tips
1. **Use the sample data**: Start with sample data to see how it looks
2. **Preview templates**: Try different templates before finalizing
3. **Test PDF export**: Make sure the PDF looks good before sharing
4. **Print test**: Print a copy to check formatting
5. **Save multiple versions**: Create different versions for different job types

## Troubleshooting

### Common Issues and Solutions

**PDF won't download:**
- Check if pop-up blockers are enabled
- Try a different browser
- Refresh the page and try again

**Form data not saving:**
- The form saves automatically as you type
- Use browser back/forward buttons to navigate
- Don't refresh the page while filling out the form

**Template not loading:**
- Clear browser cache
- Try a different browser
- Check if JavaScript is enabled

**Print formatting issues:**
- Use the print button in the preview
- Check print settings in your browser
- Ensure "Background graphics" is enabled in print settings
